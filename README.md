# Password Generator - Python 3 Tkinter GUI

This is a simple Python password generator application built using the Tkinter GUI toolkit. The application allows users to generate random passwords of varying length and complexity.

## How to Use

1. Launch the application by running the script `pw_gen2.py`.
2. Select the desired password length using the drop-down menu or type the desired length you would like, up to 2048 characters.
3. Choose the character sets to include in the password by checking the corresponding boxes. The available character sets are:
   - Uppercase letters (A-Z)
   - Lowercase letters (a-z)
   - Digits (0-9)
   - Punctuation (!@#$%^&*()_+-=[]{};':"\<>?,./|)
4. Click the "Generate Password" button to generate a random password based on the selected options.
5. The generated password will be displayed in the text box below the button.
6. If you wish to copy the password to the clipboard, click the "Copy Password" button.

## Requirements

This application requires Python 3 and the Tkinter library to be installed on your system. No additional packages or dependencies are required.
